# 异步
- 进程
- 线程


1. 在不同的场景中进程都可以来描述该场景的一个效果, 线程是进程里面的一个更小的单位, 通常多个线程配合工作构成一个进程

2. v8 运行一份 js 代码, 会创建一个进程, 从上往下执行代码, 遇到同步代码就直接执行, 遇到异步代码就会放到异步队列中, 同步代码执行完成后, 会从异步队列中取出代码执行

3. js 默认是单线程的语言: 因为 js 设定是为了做浏览器的脚本语言, 尽量少的开销用户设备的性能
