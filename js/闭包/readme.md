# 作用域链
- js 引擎在查找变量时，会先从当前作用域开始查找，如果找到就返回，如果没有找到，就从父级作用域开始查找，直到全局作用域

- 作用域链的下一级是谁，是由 outer 指针决定的

- 作用域链的作用：
    - 1. 查找变量
    - 2. 保护变量不被污染
    - 3. 闭包
- 闭包：
    - 1. 闭包的定义：
        - 闭包是指在一个函数中定义的函数，闭包可以访问到函数中定义的变量
    - 2. 闭包的作用：
- js 引擎在查找变量时，会先从当前

# 闭包
- 根据作用域链的查找规则，内部函数一定有权利访问外部函数变量。
- 一个函数执行完毕后它的执行上下文一定会被销毁
- 那么当函数 A 内部声明了一个函数 B, 而函数 B 被拿到A的外部执行时,为保证以上两个规则正常执行,A 函数在执行完毕后会将 B 需要访问的变量保存在一个集合中,并留在调用栈当中,这个集合就是闭包。

- 缺点: 内存泄漏(调用栈的可用空间越来越少)  -- 某些”干脆利落“封装
