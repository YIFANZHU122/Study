# hash
 - 在浏览器中, url 中携带了 # , # 后面的内容就是 hash 值, 浏览器中 hash 值变更是不会带来页面刷新的

原理: 监听 hashchange 事件, 当 hash 值变化时, 就会触发 hashchange 事件, 我们可以在事件处理函数中, 拿到当前的 hash 值, 然后根据 hash 值, 渲染对应的组件


# history
- 浏览器中的历史记录
- 可以通过 pushState 方法, 来添加一条历史记录， 修改url 不带来页面刷新
- 可以通过 popstate 事件, 来监听历史记录的变化，关联到了浏览器的前进后退按钮
- 可以通过 replaceState 方法, 来替换当前的历史记录
